# Reference Board & Aesthetic Audit Spec

This reference board defines the functional boundaries for the design and motion mechanics of the Product Systems Engine portfolio.

## 1. Visual & Typography Reference: Linear App / Stripe Core
* **Context:** Premium developer-centric product layouts.
* **What to Borrow:**
  * Strict monochrome dark grids with subtle `#1a1a1a` border lines.
  * Crisp execution of Monospace typography tags (`SF Mono`, `JetBrains Mono`) for technical parameters and metrics.
  * High-contrast content hierarchies ensuring semantic text is effortlessly scannable.
* **What to Avoid:**
  * Complex multi-colored radial gradients that dilute the minimal graphite tech look.
  * Over-saturation of micro-interactions that compete with the main content flow.

## 2. Interactive 3D Architecture Reference: Apple Product Unboxing / Next.js Production Showcases
* **Context:** Structural asset assembly mapped directly to scroll velocity.
* **What to Borrow:**
  * Clean, reversible 3D camera tracking that follows scroll progress smoothly without snapping.
  * Low-poly architectural wires assembling systematically as sections pass through viewports.
  * Lighting arrays confined to precise rim effects and clean emissive pulses.
* **What to Avoid:**
  * Game-style camera configurations that strip orientation control away from the user.
  * Heavy particle simulation bursts that generate memory leaks and trigger frame stutters.

## 3. Accessibility & Performance Benchmarks: GitHub Universe / Vercel Core Features
* **Context:** Highly accessible marketing experiences maintaining deep interactive depths.
* **What to Borrow:**
  * Perfect text contrast scaling over dark interactive canvas spaces.
  * Layered layout approach where hiding the `canvas` layer leaves a completely flawless semantic HTML site.
  * Inline SVG and structural representations acting as explicit network fallbacks.
* **What to Avoid:**
  * Trapping keyboard focus patterns inside interactive canvas matrices.
  * Relying purely on scroll motion to unveil essential target text layouts.
