# Narrative Arc Specification — The Product Systems Engine

This document defines the structural script for the portfolio experience, mapping the journey from a raw data signal into a fully shipped product system.

## 1. Overall Narrative Framework
The overarching story arc follows a deterministic tech lifecycle: **Signal → System Boot → Identity → Proof → Architecture → Projects → Case Study Preview → Stack → Workflow → Timeline → Contact**. The website operates as a living environment where the user's scroll input serves as the execution thread that builds, tests, and deploys digital product systems.

---

## 2. Detailed Scene-by-Scene Script

### Stage 01: Signal (Pre-initialization)
* **Narrative Concept:** The viewport opens on an uncompiled dark space. A single, precise grid-aligned point stream appears, representing the inception of a raw digital product system.
* **Interaction Flow:** As page content mounts, the signal point stream transitions into structural coordinates.

### Stage 02: System Boot (Hero Section)
* **Narrative Concept:** The raw points assemble into interconnected structural rings. The engine initializes, revealing a premium, modern technical workspace.
* **Text Layer Alignment:** The core branding headers and action-oriented CTAs ("Initiate Connection") fade into view cleanly on the semantic DOM layer.

### Stage 03: Identity (Profile Lock)
* **Narrative Concept:** The structural core slides into a stable, lateral close-up composition, splitting into multi-dimensional layers representing frontend presentation and underlying resource logic.
* **Text Layer Alignment:** Renders the profile summary, detailing how the user's Business Administration foundation enforces engineering lifecycle oversight and project management.

### Stage 04: Proof (Metric Scan)
* **Narrative Concept:** A vertical light grid scans down the structural layers. Verified node parameters light up as the engine benchmarks credibility.
* **Text Layer Alignment:** Highlights core historical metrics: 4+ production architectures, technical track leadership, and perfect web vitals.

### Stage 05: Architecture (System Assembly)
* **Narrative Concept:** The geometry tilts into an engineering perspective plane. Separate structural plates stack up sequentially to represent the application layer (Next.js components, typed structures, Supabase data connections).
* **Text Layer Alignment:** Explains the engineering choices behind reliable web system deployment.

### Stage 06: Projects (Selected Cohorts)
* **Narrative Concept:** The view tracks linearly across the structural plane, passing through 4 self-contained capsule blocks. Each capsule displays interactive preview wires.
* **Text Layer Alignment:** Interactive DOM text cards overlay each project module (Enactus Portal, AI Job Board, GDG Module, Lawyer Application).

### Stage 07: Case Study Preview (Bridge Vector)
* **Narrative Concept:** The camera moves inside a capsule node, changing the perspective into an editorial workspace code log.
* **Text Layer Alignment:** Transitions the user seamlessly from aesthetic design into the technical proof logs.

### Stage 08: Stack (Engine Capabilities)
* **Narrative Concept:** The code logs re-assemble into flat structural capability plates categorized neatly by domain.
* **Text Layer Alignment:** Displays specific framework capabilities (React, TypeScript, SQL databases) without random logo walls.

### Stage 09: Workflow (Build Pipeline)
* **Narrative Concept:** A swift data signal moves horizontally across the structural layout, lighting up sequential milestones (Briefing → Compilation → Deployment).
* **Text Layer Alignment:** Outlines the strict Discovery-to-Ship protocol.

### Stage 10: Timeline (Career Growth)
* **Narrative Concept:** The workflow signal moves along an expanded layout timeline, passing structural branch lines and commit nodes.
* **Text Layer Alignment:** Documents management highlights and frontend project milestones historically.

### Stage 11: Contact (Production Launch)
* **Narrative Concept:** The camera pulls back into a calm composition. The full system glows with a slow emissive pulse, signaling that the platform is operational and ready to convert inquiries.
* **Text Layer Alignment:** Presents the prominent final contact CTA panels linked to LinkedIn, GitHub, and email nodes.
